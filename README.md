# Lampa IPTV Plugin

IPTV плагин для [Lampa TV](https://lampa.mx) — загрузка каналов из M3U плейлиста с поддержкой EPG (программа передач).

## Возможности

- Загрузка M3U плейлиста по URL
- Группы каналов из `group-title`
- Программа передач (XMLTV / EPG)
- Список и сетка каналов
- Избранное, история просмотров, поиск
- Скрытие ненужных каналов
- **OSD мини-список каналов** при переключении кнопками CH+/CH-
- **EPG-сайдбар** с программой передач текущего канала

## Управление

| Действие | Клавиша |
|----------|---------|
| Воспроизвести канал | OK (клик) |
| Карточка канала | Долгое нажатие |
| Переключить канал | CH+/CH- (или PageUp/PageDown) |
| Открыть EPG-сайдбар | OK при видимом OSD |
| Навигация по EPG | ↑/↓ |
| Закрыть EPG-сайдбар | ← или Back |

### OSD при переключении каналов

При нажатии CH+/CH- во время просмотра появляется полупрозрачная панель с текущим каналом и соседними (до 5). Показывает название текущей передачи из EPG. Скрывается автоматически через 3 секунды.

### EPG-сайдбар

Нажмите OK при видимом OSD — справа откроется панель с программой передач. Текущая передача подсвечена с прогресс-баром, прошедшие приглушены. Листайте стрелками, закройте по Back. При переключении канала (CH+/CH-) сайдбар обновляется автоматически.

## Установка

В Lampa: **Настройки → Плагины → Добавить плагин** → вставить ссылку:

```
https://cdn.jsdelivr.net/gh/smackftw/lampa_iptv@main/dist/lampa-iptv.js
```

## Настройка

После установки в **Настройки → IPTV**:
- **M3U URL** — ссылка на плейлист
- **EPG URL** — ссылка на XMLTV (необязательно)

## Сборка из исходников
